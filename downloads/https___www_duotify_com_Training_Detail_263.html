教育訓練ASP.NET Web API 2 開發實戰
1 天
後端.NET技術類 ASP.NET Web API 2 開發實戰本次課程將帶領大家快速上手 ASP.NET Web API 2 開發框架，我們會從觀念著手， 先帶大家建立起正確的開發觀念，再透過現場的實際演練驗證自己的觀念是否正確， 用自己的雙手打造出優異的 API 服務，相信可以帶領大家在最短時間內掌握 ASP.NET Web API 2 開發框架。

此課程適合給...

  • 已有 ASP.NET MVC 開發經驗的開發人員
  • 具備 Entity Framework 基礎開發知識

上完此課程你將會...

  • 學會 ASP.NET Web API 開發方法
  • 學會正確的 RESTful API 開發觀念
  • 學會 ASP.NET Web API 各式開發技巧
  • 了解 ASP.NET Web API 完整生命週期
  • 了解各種使用 ASP.NET Web API 的陷阱與地雷

無遠弗屆的網際網路中，有越來越多的終端裝置連結著彼此，無論是瀏覽器、手機、或是物聯網裝置(IoT)， 都需要某種程度上與伺服器溝通。回首過往，你會發現有各式各樣的 API 存取方式不斷推陳出新， 以前大家可能會用 XML Web Service 來交換資料，雖然是工業標準，但由於 SOAP 協定過於繁瑣， 對於各裝置平台的呼叫方式不是特別親民。現在大家普遍接受以 RESTful (HTTP + JSON) 作為主要資料交換方式， 不但架構彈性、容易擴充，更深受廣大開發人員的擁護與愛戴，儼然成為業界標準(de-facto standard)。

作為一個 ASP.NET 開發人員，自然會選擇 ASP.NET Web API 作為主要的 API 開發框架， 因為 ASP.NET Web API 與 ASP.NET MVC 在架構極為相似，對於已經有 ASP.NET MVC 開發經驗的開發人員來說， 只要透過適當的引導，就可以在短時間內快速掌握 ASP.NET Web API 框架的精髓之處。 而本課程就是針對現有 ASP.NET MVC 開發人員所量身打造的課程，只要在一天七小時的時間內， 就可以完整掌握 ASP.NET Web API 的重要觀念與開發技巧，可以大幅減少學習與摸索的時間。

本次課程將帶領大家快速上手 ASP.NET Web API 2 開發框架，我們會從觀念著手， 先帶大家建立起正確的開發觀念，再透過現場的實際演練驗證自己的觀念是否正確， 用自己的雙手打造出優異的 API 服務，相信可以帶領大家在最短時間內掌握 ASP.NET Web API 2 開發框架。
