教育訓練JavaScript 開發實戰：核心概念篇(全新ES6改版)
1 天
網頁前端技術類 JavaScript開發實戰：核心概念篇(全新ES6改版)本課程從設計之初至今已超過 5 年，歷久不衰的教材與獨到的互動教學方式，幫助一千多名學員真正認識 JavaScript 重要的語言特性。我們在 2019 年推出 JavaScript 開發實戰：核心概念篇 (第二版) 課程，特別加入 ES6 (ES2015) 最重要的語法與技巧，除了學習重要的語言特性外，也要能跟上時代，用全新的語法改善程式碼品質、提升可讀性！

此課程適合給...

  • Web 開發人員 / 網頁設計師 / 網頁前端工程師
  • 具有基礎的程式設計概念 (寫過任何一種程式語言即可)
  • 現有 ASP.NET MVC 開發人員
  • 熟悉 HTML 語法 ( 課程中的範例，將會以 HTML 及 JavaScript 為主 )
  • 寫過 jQuery 且知道如何使用 jQuery 外掛套件 (基本 JS 開發能力)
  • 本課程不適合完全沒有 JavaScript 開發經驗的人

上完此課程你將會...

  • 了解 JavaScript 程式語言中最重要的觀念與特性
  • 了解 JavaScript 程式語言中以 原型 (Prototypal) 為主的物件導向架構

JavaScript 大師 Douglas Crockford 曾經說過：「JavaScript 是這個世界上被誤解最深的程式語言」，許多前端工程師在撰寫 JavaScript 的時候，大都只用來解決極其微小的問題，例如表單欄位驗證、基本的 DOM 操作、...等等，從來不把它當成一個「程式語言」來看待，而在不夠理解程式語言特性的情況下撰寫程式，自然也寫不出甚麼稍微複雜的應用，所以也不會刻意的去深入理解這個程式語言，因而形成一種惡性循環。

這幾年 SPA (單一頁面應用程式) 正夯，無論你用 Angular, React 還是 Vue 都一樣，最後都考驗著前端工程師對 JavaScript 的基礎能力，這並不是說你寫了 5 年、10 年就會有十足功力，如果你沒有真正理解重要的語言特性，程式碼品質肯定不會太好，寫出 Bug 的機會也會大幅增加。

有鑑於大多數前端開發者經常處於不熟悉 JavaScript 程式語言特性的狀態下，即便你想學習任何一套的 JS 框架，都不可能學的好、學的深，為此我們規劃了一個整天 7 小時的課程，希望帶領大家完整掌握 JavaScript 的精妙之處。

本課程從設計之初至今已超過 5 年，歷久不衰的教材與獨到的互動教學方式，幫助一千多名學員真正認識 JavaScript 重要的語言特性。我們在 2019 年推出 JavaScript 開發實戰：核心概念篇 (第二版) 課程，特別加入 ES6 (ES2015) 最重要的語法與技巧，除了學習重要的語言特性外，也要能跟上時代，用全新的語法改善程式碼品質、提升可讀性！

學員C

JavaScript 是一個看似亂無章法的程式語言,在保哥詳細的解說之下,許多模糊的觀念才漸漸的被導正起來,雖然這只是個開始,但總算打開認識 JavaScript的門。

學員D

課程內容紮實，著重釐清程式基礎邏輯(基礎但不簡單)。教材感覺上都是書上、網路上有寫的內容， 看完好像懂了，實際寫起來卻又不是很懂，但問題出在哪也說不出所以然。其實我們只是需要一盞明燈， 指引一個方向，照亮路面的坑洞，讓我們走的對走的順，而此堂課程達成很好的效果， 所謂「基本功練的好，寫code沒煩惱」，感謝多奇，讓我對JavaScript有新的認識。

學員E

今天上課以前，會覺得書上對於變數、物件的運作不重要。 但今天算是蠻震撼的，有種很深的罪惡感，浪費以前寫 JS 的光陰。 今天上完課後對基本功更能掌握，會再把優良部份那本書重新看一次。 很感謝保哥，這堂課真的很超值。

學員A

老師真的是充滿熱情的人, 不斷地講解也讓我們不斷的練習, 從練習中發現一些問題, 也解決一些盲點, 尤其是在物件、變數、型別那邊詳細又精闢的解說, 把很多不懂的觀念釐清, 也解開了一些在撰寫程式上的神秘事件, 還有觀念上的重建也很重要~
