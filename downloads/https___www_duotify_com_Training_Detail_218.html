教育訓練Windows Containers 微服務架構實戰
1 天
容器技術類 Windows Containers 微服務架構實戰本次課程，將帶領大家進入 Windows Containers 容器技術的殿堂，不但有容器技術的核心架構解說， 更有完整的實際操作練習，讓您在短時間內就能徹底掌握 Windows 平台下最新的容器技術， 藉此改善企業內部開發、測試、發行、佈署、維運、監控等完整的 DevOps 開發營運流程。

此課程適合給...

  • DevOps 工程師
  • 現有 .NET 開發人員
  • 企業資訊開發部門經理
  • 以網站建置為主的開發人員
  • 現有 ASP.NET Web Form 開發人員
  • 現有 ASP.NET MVC 開發人員
  • IT 系統管理或維運人員

上完此課程你將會...

  • 了解 Windows Containers 運作架構
  • 熟悉 Windows 容器生命週期與基本指令
  • 熟悉如何透過容器對應用程式進行封裝與部署
  • 能夠自行建置各種不同用途的容器
  • 能夠自動化容器管理流程
  • 能夠在正式環境部署容器

容器技術 (Container Technology) 為應用程式提供標準化的封裝、發行與部署流程， 不但讓應用程式之間可以得到更佳的隔離性，更能讓我們在不受硬體、主機、作業系統、資料中心等等限制下， 彈性的配置我們的應用程式。這幾年，容器技術發展十分迅速， 從 Docker 的興起到各大廠的不斷表態支援各式容器技術就可以看出， 這絕對是個不容小覷的明星技術。ThoughtWorks 最近幾期的技術雷達， 也都將「容器技術」視為相當重要的 IT 基礎建設。

本次課程，將帶領大家進入 Windows Containers 容器技術的殿堂，不但有容器技術的核心架構解說， 更有完整的實際操作練習，讓您在短時間內就能徹底掌握 Windows 平台下最新的容器技術， 藉此改善企業內部開發、測試、發行、佈署、維運、監控等完整的 DevOps 開發營運流程。
